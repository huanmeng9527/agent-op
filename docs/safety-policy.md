# Safety Policy

This project is intended for paper reproduction planning, source discovery, and reproducibility assessment.

Allowed uses:

- Discovering public implementation candidates
- Comparing repository structure and experiment assets
- Planning clean-room reproductions
- Checking whether a repository includes training, evaluation, config, dataset, checkpoint, and environment hints

Disallowed uses:

- Copying repositories without attribution
- Presenting someone else's code or metrics as original work
- Hiding sources in a reproduction report
- Treating an unverified repository as a faithful reproduction of the paper

All outputs include a safety note. High-risk or weak repositories should be used only as discovery leads.
